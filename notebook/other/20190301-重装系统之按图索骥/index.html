    热度0℃软件笔记本

    前言

    之前也有过一篇关于重装系统之按图索骥。恰巧今天又重装了一次系统，索性再写一次，就当是把之前写过的文章进行完善。此次选择安装的系统是windows_10_enterprise_ltsb_2016，也就是最新的企业版系统，好处在于界面上精简了很多内容，看着特别清爽，而且更新不会那么频繁，比较稳定，缺点就是精简了很多内容，有些功能得自行寻找软件替代。

    以下所有的内容，均针对于windows而言，虽然说其他平台也有很多好用的工具，但是window太方便了，对于不怎么想折腾的人来说，是极好的。

    系统设置篇

    安装好的系统，得进行一些初始化的配置，才能够让自己使用起来得心应手。

    1. 文件拓展名

      依次点击 文件资源管理器 -> 功能栏上的查看按钮 -> 勾选文件拓展名

    2. 隐藏最近访问

      依次点击 文件资源管理器 -> 功能栏上的查看按钮 -> 选项 -> 去除隐私项的两个勾选。

    3. 关闭盖子不休眠

      依次点击 全部设置 -> 显示 -> 电源与睡眠 -> 其他电源设置 -> 选择关闭笔记本计算机盖的功能 -> 关闭盖子，不采取操作。

    4. 开启自启

      有些小工具，没有开机自启功能，就需要依托系统提供的开启自启文件夹。

      将软件的快捷方式，移到C:\ProgramData\Microsoft\Windows\Start Menu\Programs\StartUp 这样所有用户开机，均能自启某些软件。

    软件安装篇

    经常有人说，360，金山，腾讯等安全管家类软件都是老流氓，建议不要去装这些，不如直接使用windows自带的防御软件，或者说装一个火绒也还挺好。

    但个人还是偏向于安装腾讯安全管家，因为对咱而言，感觉还是挺方便的，也没有太过多的受到腾讯安全管家的影响，反而帮我解决了很多麻烦。

    工具类

    1. 7-Zip 压缩/解压软件，支持格式有7z，rar，zip。
    2. Snipaste 强大的截屏软件。
    3. ScreenToGif gif录屏软件，生成gif图片。
    4. Listary 文件搜寻工具，经常用它来快速定位文件。
    5. typora 一款开源的markdown文件编辑器。
    6. JJDown 用于下载bilibili网站视频的软件。
    7. AeroTimerPlus 一款多定时的定时器软件。
    8. TeamViewer 远程软件，个人版免费。
    9. 福昕阅读器 一款pdf阅读器，体验良好。
    10. 看图王（去广告版） 查看图片的软件。
    11. UltraISO（软碟通）, 刻盘软件，u盘变成系统盘，就是用的它。
    12. clover 让文件资源管理器，变成浏览器一样，多tab页查看文件。设为英文无广告

    日常类

    1. office套装。
    2. 腾讯全家桶，包括安全管家，TIM，微信PC，QQ拼音。
    3. Chrome浏览器，当然要有良好的体验，得用点别的方法。
    4. potplayer 本地的视频播放器。
    5. 网易云音乐、百度云盘、迅雷等。
